# belt-cli

Minimalistic extendable CLI script runner, inspired by [`oclif`](https://github.com/oclif/oclif)

## Install

```sh
$ npm install --global belt-cli
$ yarn global add belt-cli
```

## Usage

```sh
$ belt --install path/to/module
$ belt --list
$ belt <command> <args>
```
